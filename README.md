# 📊 Survey Data Dashboard

An interactive **Streamlit dashboard** for analyzing survey data, exploring user satisfaction, and generating actionable insights.

## 🚀 Overview

This project visualizes survey responses to help understand customer behavior and satisfaction trends. Users can interact with the data through filters and visualizations to uncover meaningful insights.

## ✨ Features

* 📋 **Dataset Preview** – View raw survey data
* 📊 **Average Satisfaction Score** – Quick KPI overview
* 📈 **Recommendation Analysis** – Visual breakdown of user recommendations
* 🔍 **Filter by Gender** – Dynamic data filtering
* 💡 **Key Insights Section** – Automatically generated insights

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib

survey-data-dashboard/

## ▶️ Run Locally

```bash
git clone https://github.com/SthembileKubheka/survey-data-dashboard.git
cd survey-data-dashboard
pip install -r requirements.txt
streamlit run app.py
```
